寓言

城南旧城有断墙犹存，余近日每每路过，不免胆战。传闻断墙下曾有一洞，洞内曾有一物，名曰大黄，常叫嚣曰：矢乎矢乎，世之美味也哉，吾虽行千里亦必食之。
一日，景阳冈上偶遇大吊虫，大黄思虑，若得此为盟友，则吾于此百兽之间，富贵功名如探囊取物，唾手可得耳。遂拿出了伊珍藏了三年的矢，与之虎，曰：“嗟，请来食”，说完将脖子扭几扭，实在是标志极了。
余不一一，盖因本文主角已惨死虎口之下。

拿破仑

我们从小习惯了“没有XX就没有XX”，“XX是XX的鲜血染成的”，这样的句式，然后冠以知恩图报的思想，仿佛不报恩便是坏。
历史上其实恩格斯本人也说过：

恰巧拿破仑这个科西嘉岛人做了被战争弄得精疲力竭的法兰西共和国所需要的军事独裁者，假如不曾有拿破仑这个人，那么他的角色是会由另一个人来扮演的。

这句句式不是空话，其实也可以套用在很多道理上的。
即便没有先辈的鲜血，也将由XXX。
因为，我相信人性都是有追求的，内里深处都有最起码的自由快乐的渴望，就像美国黑奴几十年，近200年的历史，不是能够走到今天吗？
难道因为岳飞抗金胜利使人就应该对其感恩戴德唯命是从供养其后代吗？
《七武士》结尾写得好，英雄明白，既然兴亡百姓皆苦，那么面对拯救英雄又怎能奢求被救者感激涕零之外更多呢？
华盛顿干完两届后回家务了农。

雪崩的时候没有一片雪花觉得自己有责任

现代人写文章喜欢引用来讲道理。
但是，让我们想一想，雪崩了，难道雪花有责任吗？
我称如上为一种文字工作者的诡辩，营造一种悲剧(英雄则更佳)的景象，去共鸣读者的大脑，也就是使读者免于思考但因共鸣某种感受而采纳结论却选择性的去认知文字涵义。
